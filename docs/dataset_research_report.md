# 多肽合成数据集调研报告

## 1. 现有数据集分析

### 1.1 项目内部数据集

#### A. final_purity_yield_literature.csv
- **记录数**: 126条
- **数据来源**: 已发表的文献数据
- **包含字段**:
  - 序列信息 (sequence)
  - 纯度 (purity_pct)
  - 收率 (yield_pct)
  - 合成条件 (condition_summary)
  - 拓扑结构 (topology)
- **数据特点**:
  - 真实实验数据
  - 包含文献引用信息
  - 覆盖多种肽类型（淀粉样蛋白、硫酯肽等）

#### B. synthesis_data.csv
- **记录数**: 约2000+条
- **数据来源**: 自动流动肽合成仪(AFPS)实时数据
- **包含字段**:
  - 氨基酸类型 (amino_acid)
  - 偶联试剂 (coupling_agent)
  - 温度参数 (temp_coupling, temp_reactor_1)
  - 流速 (flow_rate)
  - 色谱峰参数 (area, height, width, diff)
- **数据特点**:
  - 步骤级详细数据
  - 实时合成监控数据
  - 适合训练步骤级预测模型

## 3. 数据集整合策略

### 3.1 数据增强方案

```python
# 建议的数据增强方法
1. 序列级增强:
   - 氨基酸替换（保持性质相似）
   - 序列片段重排
   - 长度变化模拟

2. 条件级增强:
   - 温度扰动 (±5°C)
   - 时间扰动 (±10%)
   - 试剂浓度变化

3. 合成结果增强:
   - 基于现有分布的采样
   - 噪声注入（高斯噪声）
```

### 3.2 特征工程建议

#### 新增特征（基于文献调研）:
1. **氨基酸物理化学性质**:
   - 疏水性指数 (Kyte-Doolittle)
   - 等电点 (pI)
   - 分子量
   - 侧链体积

2. **序列结构特征**:
   - α-螺旋倾向性
   - β-折叠倾向性
   - 转角形成概率

3. **合成难度指标**:
   - 氨基酸偶联难度评分
   - 聚集倾向指数
   - 二级结构干扰度

## 4. 模型调优建议

### 4.1 数据预处理

```python
# 建议的数据预处理流程
1. 数据清洗:
   - 去除异常值 ( purity > 100% 或 < 0%)
   - 处理缺失值
   - 标准化序列格式

2. 特征标准化:
   - Z-score标准化
   - 类别变量编码
   - 序列特征向量化

3. 数据分割:
   - 训练集: 70%
   - 验证集: 15%
   - 测试集: 15%
   - 按序列相似性分层
```

### 4.2 模型改进方向

#### A. 集成学习
- 结合多个基学习器
- 使用Stacking策略
- 集成文献数据和实验数据

#### B. 迁移学习
- 利用预训练的蛋白质语言模型
- 从大规模肽数据集中学习表示
- 微调至合成预测任务

#### C. 多任务学习
- 同时预测纯度和收率
- 共享序列表示层
- 任务特定的输出层

### 4.3 评估指标优化

```python
# 建议的评估指标
1. 主要指标:
   - RMSE (均方根误差)
   - MAE (平均绝对误差)
   - R² 决定系数

2. 业务指标:
   - 纯度预测准确率 (±5%范围内)
   - 收率预测准确率 (±10%范围内)
   - 合成难度分级准确率

3. 稳定性指标:
   - 交叉验证稳定性
   - 不同批次数据一致性
```
