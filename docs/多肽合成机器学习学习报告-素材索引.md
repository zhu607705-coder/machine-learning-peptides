## 4 关键数字速查表

### synthetic MLP

- 数据总量：1080
- 场景数：6
- split：750 / 162 / 168
- hidden size：8
- learning rate：0.12
- L2：0.0015
- validation combined RMSE：4.0063
- test combined RMSE：4.1082
- 并行优化运行时间：约 0.814s

### peptimizer 真实数据

- 数据源：`learningmatter-mit/peptimizer`
- 样本数：12600
- split：8907 / 1791 / 1902
- grouped split：`serial`
- 最优架构：`gru_residual_small`
- embedding dim：16
- sequence hidden：32
- numeric hidden：32
- trunk hidden：96
- dropout：0.12

### comprehensive workflow

- 初始样本：120
- 清洗后样本：108
- 异常值删除：11
- baseline purity RMSE：20.042
- baseline yield RMSE：36.871
- embedding purity RMSE：21.034
- embedding yield RMSE：37.810
- embedding joint accuracy：0.056

### Bayesian hierarchical

- literature 总记录：135
- `sequence + isolated yield`：29
- 研究来源数：5
- pooled mean isolated yield：30.53
- I²：0.986
- Grouped Ridge RMSE：47.218
- Grouped MixedLM RMSE：48.428
- Grouped Bayesian RMSE：33.070
- posterior ICC：0.897
- mean interval width：88.945

### Fmoc SOP subset

- strict Fmoc-like 行数：66
- 来源：`amyloid_hydrophobic_2008`、`teabags_fmoc_2021`、`trityl_anchor_2013`
- `yield_stage`：`isolated=16`，`crude=50`
- `purity_stage`：`final_product=14`，`crude_hplc=52`
- yield 相对均值基线有明显改善
- purity 仍受标签语义混合限制

### Fmoc SOP + protein embedding

- checkpoint：`facebook/esm2_t6_8M_UR50D`
- unique sequence count：63
- raw embedding dim：320
- PCA dim：8
- purity RMSE：`29.281`
- yield RMSE：`30.867`
- 结论：embedding 在 SOP 收缩子集上带来小幅但真实的额外增益

### Wide-task embedding comparison

- checkpoint：`facebook/esm2_t6_8M_UR50D`
- embedding 搜索迭代数：2
- baseline purity RMSE：`20.042`
- embedding purity RMSE：`21.034`
- baseline yield RMSE：`36.871`
- embedding yield RMSE：`37.810`
- 结论：在宽泛文献任务上，embedding 未改善总体 RMSE，标签语义异质性仍是主导因素
